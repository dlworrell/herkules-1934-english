# Pre-EDT Work Archive

This directory preserves HERKULES project work created before the repository was reorganized around Engineering Docs Toolkit (EDT).

Use this archive for older notes, draft translations, glossary experiments, indexing notes, and earlier build experiments that may still be useful later.

Active canonical project material belongs in the main project folders such as `source/`, `reference/`, `reports/`, and `output/`.

Material may be promoted from this archive into the active project tree after review. Preserve provenance in commit messages and notes when doing so.
