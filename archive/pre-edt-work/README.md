# Pre-EDT Work Archive

This directory preserves HERKULES project material created before integration with Engineering Docs Toolkit (EDT).

Nothing in this archive is considered canonical going forward. The canonical source tree will be built around EDT project configuration, source material, reference assets, reports, and generated outputs.

Material should be moved here when it is obsolete, exploratory, superseded, or historically useful but no longer part of the active build path.

Useful material should be migrated deliberately into the active structure over time, with provenance retained.

## Migration policy

- Preserve editorial work before deleting anything.
- Keep source-derived notes, drafts, glossaries, concordances, and page notes if they may help reconstruction.
- Archive old one-off scripts and layout experiments rather than deleting them immediately.
- Delete only generated junk that is reproducible and has no historical value.
- Record significant moves in `MIGRATION.md`.
