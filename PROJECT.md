# Project Charter

## Purpose

This repository is the working source tree for the HERKULES 1934 English technical edition.

The project treats the manual as a maintainable engineering publication rather than a one-off translation.

## Goals

1. Preserve original section numbering and engineering meaning.
2. Translate Swedish workshop prose into clear English shop-manual language.
3. Maintain terminology in a shared glossary.
4. Build indexes and concordances from the source files.
5. Produce repeatable PDF, HTML, DOCX, and EPUB outputs.

## Source policy

Do not commit source scans or PDFs unless redistribution rights are confirmed.

Private local source material belongs outside Git or in ignored paths.
