# EDT Deployment Plan

This repository will consume Engineering Docs Toolkit as the reusable document engine.

## Immediate goals

1. Preserve previous HERKULES work in `archive/pre-edt-work/`.
2. Keep active translation and reference work in the canonical folders.
3. Use `edt-project.yml` as the project manifest.
4. Add EDT commands incrementally after the repository layout is stable.

## Canonical active folders

- `source/swedish/`
- `source/english/`
- `reference/glossary/`
- `reference/indexes/`
- `reference/concordance/`
- `reference/tm/`
- `reports/`
- `output/`

## Archive migration rule

Move old material into the archive first. Promote it back into active folders only after review.
